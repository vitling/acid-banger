# Endless Acid Banger 2.0

An algorithmic human-computer techno jam

## Requirements

- Install NodeJS
- WebMIDI requires a supported browser (currently, Chrome)

## Prerequisites

1. Clone the repository
2. Run `npm i`

## Usage

1. `npm run dev`
2. Browse to http://localhost:9000/
